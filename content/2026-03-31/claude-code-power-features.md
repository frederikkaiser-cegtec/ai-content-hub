## Kernaussagen
- Boris Cherny (Claude Code Creator) hat seine 15 meistgenutzten Power Features veroeffentlicht
- Claude Code ist mehr als ein Chat-Interface — es hat ein vollstaendiges Tool-Oekosystem
- Die meisten Nutzer kratzen nur an der Oberflaeche und nutzen es wie ein besseres ChatGPT
- Hidden Features ermoeglichen tiefere Codebase-Integration, automatisierte Workflows und Team-Kollaboration
- Der Gap zwischen Casual-Nutzung und Power-Nutzung ist bei Coding-Agents besonders gross

## Relevanz fuer Marketing
Fuer Marketing-Teams, die AI Agents fuer Automatisierung oder Website-Entwicklung nutzen, zeigt dieser Artikel das Potenzial von Coding Agents. Wer nur promptet statt konfiguriert, laesst 80% der Moeglichkeiten liegen.
